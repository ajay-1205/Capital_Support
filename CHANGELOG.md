# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-17

### Added
- ✨ Initial release of Capital Support
- 🔐 User authentication with JWT tokens
- 👥 Complete customer management (CRUD)
- 💰 Loan creation and tracking
- 📊 Payment recording with automatic calculations
- 🎨 Modern, responsive web interface
- 📱 Mobile-friendly design
- 🔍 Real-time customer search
- 📧 Email verification system
- 🔄 CORS support for multi-origin deployments
- 🐳 Docker support
- 📦 Render deployment ready
- 📚 Comprehensive documentation

### Features
- Customer profile management
- Loan term and interest rate tracking
- Automatic interest calculations
- Payment history tracking
- Principal and interest breakdown
- User authentication and authorization
- Database persistence
- RESTful API with Swagger documentation
- Beautiful dashboard UI

### Technical
- FastAPI backend with async support
- SQLAlchemy ORM with PostgreSQL/SQLite support
- Pydantic data validation
- FastAPI Users for authentication
- Uvicorn ASGI server
- Vanilla JavaScript frontend (no dependencies)
- CSS3 custom design system

## How to Report Changes

- 🐛 **Bug**: Use GitHub Issues with label `bug`
- ✨ **Feature**: Propose in GitHub Issues or Discussions

- 🎨 **UI/UX**: Create issue with screenshots

## Versioning Scheme

- **Major** (X.0.0): Breaking changes, major features
- **Minor** (0.X.0): New features, backwards compatible
- **Patch** (0.0.X): Bug fixes, small improvements
